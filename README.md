## Installation

### RubyGems

```
$ gem install fluent-plugin-arrow
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-arrow"
```

And then execute:

```
$ bundle
```

## Configuration

You can generate configuration template:

```
$ fluent-plugin-config-format formatter arrow
```
